AtomicRedTeamCrawler:
    Thuộc tính:
        url: đường dẫn đến trang github của atomic redteam
        techique: lưu các techique crawl được dưới dạng map
        data: raw data lưu dưới dạng đoạn văn
    Phương thức:
        webCrawler(): phương thức này sẽ crawl dữ liệu từ trang github của atomic redteam và lưu dữ liệu vào biến data và technique
ExcelExporter:
    Thuộc tính:
        data: dữ liệu cần lưu xuống file excel lấy từ AtomicRedTeamCrawler.getData()
        outputFilePath: đường dẫn đến file excel để lưu dữ liệu
    Phương thức:
        export(): phương thức này sẽ lưu dữ liệu xuống file excel
MitreATTACKChecker:
    Thuộc tính:
        url: đường dẫn đến trang chủ Mitre ATT&CK
        techique: lưu các techique crawl được dưới dạng map
    Phương thức:
        coverageRate(): phương thức này sẽ đối chiếu data từ atomic redteam với nội dung trên chính trang chủ Mitre ATT&CK và tính tỉ lệ bao phủ
            2 tham số truyền vào là các technique từ atomic redteam vào Mitre
        countMissingTechnique(): tìm các technique chưa được bao phủ
ChartGenerator:
    Thuộc tính:
        rate: kết quả tính tỉ lệ bao phủ lấy từ class MitreATTACKChecker
        missingTechnique: các technique chưa được bao phủ, lấy từ class MitreATTACKChecker
    Phương thức:
        generateChart(): phương thức này sẽ tạo biểu đồ từ kết quả tính tỉ lệ bao phủ
Main:

        main(): thực thi các câu lệnh
